Setting Up A Workspace

Various tools to set up a coding workspace

Posted on May 6, 2021 · < 1 minute read

Code editor | Terminal | Other helpful tools | Hardware specs

Here are some of the software I currently use for my everyday coding needs.

Code editor

  • Atom – text editor
  • Nv Dark – UI theme
  • Nv Dark – Syntax theme
  • r-exec – send R code to various R consoles
  • language-r – language description and snippets for R

    This package is currently unmaintained, but I personally prefer the syntax highlighting of this package compared to atom-language-r

Terminal

  • Terminal – built-in MacOS
  • Duotone Dark – macOS terminal theme

Other helpful tools

  • RSwitch – lightweight menubar utility for macOS to seamlessly switch between several installed R versions
  • Inkscape – open-source vector graphic software
  • Tower – powerful git client for Mac and Windows

    Included for free in the GitHub Student Developer pack with a valid student email address!

Hardware specs

  • MacBook Pro 2020
  • macOS Big Sur (version 11.3.1)
